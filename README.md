# Text-to-SQL-App
A Text-to-SQL web application that translates natural language into automated database queries, enabling users to extract analytics using plain English.

This project bridges the gap between non-technical users and complex data by leveraging local Large Language Models (LLMs) to write and execute SQL code dynamically. It ensures data privacy and zero API costs by running the AI models entirely locally.

✨ Features
Natural Language Processing: Ask questions about your data in everyday language.

Automated SQL Generation: The app writes the correct SQL syntax based on your database schema.

Real-time Execution: Instantly runs the generated query and displays the results.

Privacy-First: Uses local AI models, meaning your database schema and data never leave your machine.

Transparent Workflow: Displays the AI-generated SQL alongside the final data results for easy auditing.
